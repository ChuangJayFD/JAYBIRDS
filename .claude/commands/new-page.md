以下の要件で新しいHTMLページを作成して。
CLAUDE.mdのデザインシステムとルールに必ず従うこと。

ページ名・目的: $ARGUMENTS

## 必須チェックリスト（全項目必須）
- [ ] OGPメタタグ完備（og:title, og:description, og:image, og:url）
- [ ] twitter:cardメタタグ完備
- [ ] JSON-LD構造化データ
- [ ] canonical, hreflang（ja/en/zh-TW/ko）
- [ ] GA4トラッキングコード（G-EXHV880LX0）
- [ ] color-scheme: dark を <head> に明示
- [ ] background-color を body に hardcoded hex で指定
- [ ] カラーはhardcoded hexのみ（CSS変数・紫系禁止）
- [ ] Cormorant Garamond / Noto Sans JP フォント（Google Fonts）
- [ ] OGP画像URLに ?v=1 キャッシュバスト
- [ ] レスポンシブ対応（モバイルファースト）
- [ ] 既存ページのヘッダー・フッターナビゲーションと統一

## 作成後に自己チェック
上記チェックリストを全項目確認して、問題があれば自分で修正してから完了を報告して。
